👕 PerfectFit: AI-Driven Apparel Matching & Geospatial Discovery

PerfectFit, an AI-powered system designed to modernize how users discover and purchase clothing. The platform utilized computer vision to analyze user-uploaded T-shirt images and matched them with inventory from nearby physical and online retailers.

🚀 Key Features

1. Automated Attribute Extraction: Implemented computer vision techniques to extract clothing dimensions, landmarks, and key features directly from user-uploaded images.
2. Intelligent Recommendation Engine: Built a backend system that identified highly similar items and ranked them based on style, enabling users to find desired items without manual searching.
3. Geospatial Retail Integration: Integrated a search logic that returned up to 10 nearby stores ranked by proximity and real-time item availability.
4. Privacy-First Data Handling: Engineered a secure pipeline with automatic image deletion to ensure user privacy during the computer vision extraction process.

🏗️ Technical Stack

1. Frontend: React, TypeScript 
2. Backend: Node.js, Express.js 
3. Machine Learning: TensorFlow (Computer Vision) 

🛠️ Project Impact

1. Streamlined Discovery: Reduced the friction of finding specific apparel by translating visual data into actionable retail locations.
2. Technical Versatility: Demonstrated proficiency across the full development lifecycle, from training computer vision models to building responsive web interfaces.
